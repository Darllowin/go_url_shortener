# Go URL Shortener

Простой сокращатель ссылок, написанный на Go с использованием только стандартной библиотеки (без сторонних фреймворков).

## Особенности
- **Clean Architecture**: Разделение на слои (хранилище, бизнес-логика, транспорт).
- **In-Memory Storage**: Высокая скорость работы благодаря хранению данных в оперативной памяти с защитой от гонки данных (`RWMutex`).
- **Standard Library**: Используются только пакеты `net/http`, `encoding/json` и новый роутинг из Go 1.22.
- **Статистика**: Отслеживание количества переходов по каждой ссылке.

## Структура проекта
- `internal/storage` — логика хранения данных и интерфейс хранилища.
- `internal/handler` — HTTP-хендлеры и DTO (объекты передачи данных).
- `internal/shortener` — генератор уникальных коротких ключей.
- `cmd/server` — точка входа и инициализация приложения.

## Как запустить

### Требования
- Go 1.22 или выше

### Запуск
```bash
go run cmd/server/main.go
